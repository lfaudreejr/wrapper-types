# Wrapper-Types

Wrapper-Types is a suite of Abstract Data Types. IO, Task, Identity currently supported.

## Installation

```bash
npm install wrapper-types
```

## Usage

```javascipt
const { IO, Identity, Task } = require('wrapper-types')
```
